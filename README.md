# handwritten-digit-recognition
主要功能与使用
- 绘制并识别：在左侧画布绘制数字，点击“识别”显示预测结果与各类别概率。
- 随机 MNIST：点击“随机MNIST”加载并显示一张 MNIST 测试样例，若已加载模型会自动预测并显示真实标签。
- 训练模型：选择识别方式（CNN 或 HOG+SVM），设置轮数后点击“训练模型”开始训练（在后台线程运行）。
- 保存/加载模型：训练后会保存到默认文件名，可通过“加载模型”按钮载入。
- 反馈与再训练（仅对 CNN）：若预测错误，点击“反馈”并输入正确标签；之后点击“使用反馈数据再训练 (CNN)”对现有 CNN 模型做增量训练并保存。

常见问题与排查
- 无法导入 PyQt5：确认已在当前 Python 环境安装 PyQt5。
- TensorFlow 占用 GPU/安装问题：可通过禁用 GPU；根据环境安装合适的 TF 版本。
- 模型加载失败：确认 mnist_cnn.h5 或 mnist_svm.pkl 存在且格式正确。
- GUI 停顿：训练在后台线程执行，若仍卡顿请检查系统资源与 TensorFlow 日志。
